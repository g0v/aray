# aray

## Features

- 使用者
  - 可以瀏覽專案
  - 可以瀏覽其他使用者
  - 可以創建專案變成坑主
- 坑主/共同坑主
  - 可以新增專案任務
  - 可以給予其他使用者貢獻時數
  - 可以新增專案共同坑主

## GetStarted

```bash
# docs/React.md
npm i
npm start
# open localhost:3000
```
