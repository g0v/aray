# aray

## Features

- Users can:
  - View projects
  - View other users
  - Create new project and become a project organiser
- Pit owners/Pit co-owners can:
  - Create new tasks
  - Give contribution hours to other users
  - Create new projects with a pit co-owner

## GetStarted

```bash
# docs/React.md
npm i
npm start
# open localhost:3000
```
